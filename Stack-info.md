 * a list of useful commands used by stack
   * `stack runghc` ^= `runhaskell / runghc`
   * `stack install <pgkname>` ^= `cabal install <pgkname>`
   * `stack build` ^= `cabal build`
   * `stack ghci` ^= `ghci` 
   * `stack exec <executable>` ^= `./<executable>`
   * `stack test` ^= `cabal test`
